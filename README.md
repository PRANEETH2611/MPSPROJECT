## Cloud-Based Intelligent Incident Prediction and Automated Resolution System
### 📌 Overview

    This project focuses on monitoring cloud systems to detect unusual behavior, 
    predict possible issues, and automatically resolve common problems and  
    to reduce downtime and improve system reliability.

### 🎯 Objectives
    Monitor cloud system performance
    
    Identify early signs of system issues
    
    Automatically resolve common incidents

### 🛠️ Key Features

    Continuous system monitoring
    
    Issue detection and prediction
    
    Automated corrective actions
    
    Simple status dashboard

### 💻 Technologies

    Python
    
    Cloud Platform (AWS / GCP or simulated)
    
    Monitoring & data analysis tools
    
    Basic dashboard

### 🚀 Usage

    Clone the repository
    
    Install dependencies
    
    Copy .env.example to .env and set optional API keys
    
    Run python run_all.py for the backend + frontend development stack
    
    Run monitoring and prediction modules
    
    View results on the dashboard

### 🆕 Added Utilities

    One-command local launcher: python run_all.py
    
    Structured remediation planning API: GET or POST /api/resolution-preview
