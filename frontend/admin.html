Quick Overview

Welcome to the AIOps Admin Portal. This area is for system administration tasks like managing users, reviewing login activity, tuning alert thresholds, and validating backend health.

Dashboard: High-level stats (users, recent logins, uptime, DB status).
User Management: Add/delete users and assign roles.
System Health: Check MongoDB, data file status, and server metrics (CPU/RAM/uptime).
Configuration: Update CPU/memory/latency thresholds used by monitoring and alerts.
Login History: Review recent logins tracked in MongoDB.
Audit Logs: See admin actions (user changes, config updates, exports, test emails).
Alert Settings: Verify email alerting and review alert history.
Tip: If something doesn’t load, restart the backend and hard refresh the browser (Ctrl+F5).
